VA|PREVENTION

I served as the project software engineer for 15 months, where I took on the responsibility of developing the entire application from the ground up. I conceived and developed a comprehensive web application with a scientific foundation aimed at preventing type 2 diabetes through the use of a virtual assistant. Additionally, I configured the online web server and database, managing environment settings to ensure secure and efficient application interactions. In this project, I worked with a team of 16 researchers from five higher education institutions, gaining experience in a dynamic, team-based setting. This helped me improve my skills in coordination, communication, and adapting to changing project requirements, while contributing to the successful completion of key research goals.

Roles in the Project

My role involved not only writing clean, efficient code but also ensuring the application’s scalability, security, and performance, all while collaborating closely with the team to meet the project’s goals. My responsibilities included:

  • Software Developer: Designing and implementing both the front-end and back-end of the application using PHP, Laravel, HTML, CSS, and JavaScript, ensuring seamless integration between server-side logic and dynamic user interfaces.
  • Full-Stack Developer: Developing the entire application, working across the full technology stack to deliver a cohesive, functional product.
  • System Architect: Structuring the application’s architecture following the MVC pattern, ensuring scalability and maintainability for future iterations.
  • Database Administrator: Designing and managing the relational database schema, focusing on data integrity, optimization, and scalability.
  • DevOps Engineer: Handling server configurations, deployment pipelines, and the overall hosting environment to ensure secure and scalable infrastructure.
  • UI/UX Designer: Crafting user-centered workflows and interactive designs, focusing on accessibility for older adults, ensuring a smooth and intuitive experience.
  • Quality Assurance (QA) Engineer: Conducting thorough usability testing and debugging to refine the application’s interface navigation and improve overall performance.
  • Security Specialist: Implementing robust security features such as data encryption, role-based access controls, and secure transmission protocols to protect user data.
  • Documentation Specialist: Creating comprehensive technical guides, instructional videos, and a scientific poster to support users and developers.
  • Support Engineer: Providing ongoing maintenance, troubleshooting, and user support, ensuring smooth operation and user satisfaction throughout the lifecycle of the application.

Project Overview

This project focused on the development, implementation, and evaluation of the web application for the VA|PREVENTION project, a digital intervention aimed at preventing Type 2 Diabetes (T2D) through personalized support and behavior change strategies. With T2D being a growing global public health concern, the application offers a modern solution to the challenges of traditional prevention programs by providing accessible, consistent, and engaging support to at-risk individuals.

The application’s core innovation is the integration of a virtual assistant named Ema, which I developed to deliver motivational guidance and personalized support to users. Ema interacts dynamically with users in Portuguese, adapting dialogs based on their responses and health data, ensuring a highly tailored experience. The assistant operates across three distinct phases of the intervention: Assessment, Follow-Up, and Maintenance, providing step-by-step guidance tailored to users' progress and goals.

My Contributions

  • Application Development: Designed and implemented the web-based platform using PHP, Laravel, and other web technologies. Focused on a user-centered design approach to ensure accessibility and engagement, particularly for older adults.
  • Virtual Assistant Integration: Developed Ema with 2D animations, synchronized lip movements, and dynamic dialogs using Google Cloud Text-to-Speech. Created over 4,000 animations for natural, lifelike interactions.
  • Behavior Change Techniques: Incorporated strategies with gamification and interactive features to motivate sustainable lifestyle changes.
  • Usability Testing: Conducted iterative testing to refine the interface, navigation, and accessibility.
  • Deployment and Maintenance: Configured a secure remote server and implemented a continuous integration and deployment pipeline.
  • Supplementary Materials: Produced technical guides, training videos, and a scientific poster for workshops and conferences such as LASIGE Workshop 2024.

Key Features of VA|PREVENTION

The application includes the Virtual Assistant Ema, Educational Menus, a BackOffice Dashboard, a Support Social Network, and Personalized Alerts. Each component plays a role in supporting T2D prevention and promoting sustainable lifestyle changes.
