Introduction

This project report presents the development, implementation, and evaluation of the VA|PREVENTION web application, a digital intervention aimed at preventing Type 2 Diabetes (T2D) through personalized support and behavior change strategies. With T2D as a growing public health concern, the application provides accessible, consistent, and engaging support through the virtual assistant, Ema.

Confidential Project

This project is confidential due to its focus on health, requiring strict data protection and ethics compliance. Access is restricted, and ethics committee approval is necessary to ensure user rights, data security, and adherence to healthcare standards.

Project Collaboration and Innovation

The project was developed in collaboration with experts across various fields, including Sports Science, Nutrition, Computer Science, and Medicine. VA|PREVENTION incorporates behavior change techniques, gamification, and interactive features to create a supportive environment for users.

Key Features of VA|PREVENTION

The application includes the Virtual Assistant Ema, Educational Menus, a BackOffice Dashboard, a Support Social Network, and Personalized Alerts. Each component plays a role in supporting T2D prevention and promoting sustainable lifestyle changes.

Project Development Process

The project was organized into three phases: planning, refinement based on user feedback, and finalization. Technologies used include PHP, Laravel, and user-centered design methodologies, with a focus on creating a web application for accessibility and versatility.

Interaction with Virtual Assistant Ema

Ema, a 2D avatar, interacts with users across evaluation, follow-up, and maintenance phases. Ema provides personalized guidance, dynamically adjusts interactions, and offers motivation, enhancing the user’s experience and support throughout the behavior change process.

Phases of Intervention

The Evaluation phase identifies user behaviors, the Follow-up phase offers guidance for improvement, and the Maintenance phase helps sustain positive changes. Users can also reevaluate their progress to ensure continuous support and adaptation.

Supplementary Features

Educational Menus provide information on nutrition and physical activity, while the Support Social Network fosters community among users, encouraging behavior change and sustainability.

Usability Testing and Feedback

Feedback from diverse users, including older adults, informed improvements in the interface, navigation, and accessibility. Testing cycles ensured a user-centered approach and effectiveness throughout development.

Technical Development: Ema’s Animation

Ema’s animations were generated by synchronizing visemes with audio, using Google Cloud Text-to-Speech. The result was realistic and synchronized speech animations, enhancing the natural interaction experience for users.

Deployment and Server Configuration

Deployed on a secure remote server, the application offers high availability and scalability. Continuous integration and Git version control practices facilitate efficient updates, ensuring optimal performance and security for users.

Supplementary Materials

To support the project, technical guides, an instructional video, a glossary, and a scientific poster were created. These materials promote understanding, proper functioning, and alignment for the team and users.

VA|PREVENTION

VA|PREVENTION offers an innovative approach to T2D prevention and has the potential to positively impact public health by encouraging at-risk individuals to adopt healthier habits. This project provides a foundation for future development, aiming to advance health promotion and quality of life.
