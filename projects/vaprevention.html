Introduction

This project report presents the development, implementation, and evaluation of the VA|PREVENTION web application, a digital intervention aimed at preventing Type 2 Diabetes (T2D) through personalized support and behavior change strategies. With T2D as a growing public health concern, the application provides accessible, consistent, and engaging support through the virtual assistant, Ema.

Roles in the Project

  • Software Developer: Designed and implemented both the front-end and back-end of the application using PHP, Laravel, HTML, CSS, and JavaScript.
  • Full-Stack Developer: Developed the entire application, integrating server-side logic with dynamic and responsive user interfaces.
  • Database Administrator: Designed and managed the relational database schema, ensuring data integrity and scalability.
  • DevOps Engineer: Configured the server, managed deployment pipelines, and ensured secure and scalable application hosting.
  • System Architect: Designed the application architecture following the MVC pattern, enabling scalability and maintainability.
  • UI/UX Designer: Created user-centered workflows and interactive designs for a seamless user experience, with accessibility for older adults.
  • Quality Assurance (QA) Engineer: Conducted usability testing and debugging to improve interface navigation and application performance.
  • Security Specialist: Implemented data encryption, role-based access controls, and secure transmission protocols to protect user data.
  • Documentation Specialist: Created technical guides, instructional videos, and a scientific poster to support users and future developers.
  • Support Engineer: Provided ongoing maintenance and resolved user-reported issues for a smooth application experience.

Key Features of VA|PREVENTION

The application includes the Virtual Assistant Ema, Educational Menus, a BackOffice Dashboard, a Support Social Network, and Personalized Alerts. Each component plays a role in supporting T2D prevention and promoting sustainable lifestyle changes.
