Project Title 1

I gained expertise in parallel and concurrent programming, developing skills to write and interpret complex programs for multicore, GPU, and distributed environments. My experience spans various programming models and techniques, including embarrassingly parallel tasks, parallel divide-and-conquer, and GPU programming for intensive computations. I worked with synchronization primitives, dynamic load balancing through work-stealing, and granularity control for managing irregular parallelism. They also covered distributed memory programming, message-passing with the Actor model, and large-scale data processing with Map-Reduce. Additionally, I focused on debugging and verifying concurrent programs, ensuring efficiency and correctness in environments where tasks execute simultaneously or interleave..

Thymio robot project for visually impaired

The objective of this work was to explore Human-Robot Interaction (HRI) by creating a Maze Navigation Game using a Thymio robot for visually impaired users. For the project it was develop an accessible smartphone app with auditory and vibrotactile feedback, enabling navigation through voice commands, buttons, and gyroscope controls.

Roman Theatre - Blender Project

The objective of this project is the virtual recreation of a historical monument—the Roman Theater of Lisbon—as well as a one-minute animated tour of it.
The project was completed in Blender, with the theater's structure/foundation modeled manually and enhanced with some models imported from StackBlitz and Mixamo to create a more immersive and realistic environment.
